# game-engine: Demo game engine concept

IDK. I wanna make a roguelike from scratch

## Installation

### Windows

I use Visual Studio with my CMakeLists.txt to edit and build this project on Windows.

### Linux

I use CMake and GNU C++ Compiler (`g++`) to build this project on Linux.

To build from current directory (`/game-engine`):

```bash
cmake -B build && cmake --build build
```

To run from current directory (`/game-engine`):

```bash
build/bin/main
```
